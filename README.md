Este projeto consiste em uma pequena aplicação CRUD (Create, Read, Update, Delete) de produtos, desenvolvida com foco em aprendizado e boas práticas no backend.

A aplicação permite realizar o cadastro, listagem, atualização e exclusão de produtos, utilizando uma API REST.

#### Tecnologias Utilizadas: ####

Node.js – Ambiente de execução JavaScript

Express – Framework para criação da API REST/endpoints

Sequelize – ORM para manipulação do banco de dados

MySQL – Banco de dados relacional

dotenv – Gerenciamento de variáveis de ambiente

Nodemon - Reinicia automaticamente o servidor Node quando acontece alguma alteração nos arquivos.

#### Comandos para rodar: ####
```npm install``` <br>
```npm start```
